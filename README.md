# Catching Game

A simple HTML5 game built with Phaser 3 where players catch incoming soccer balls by tapping the screen.

## How to Play
- Tap the screen to move the gloves and catch the ball.
- You score points for every catch.
- You have 3 lives. The game ends if you lose all lives or the time runs out.

## How to Run
1. Open a terminal in the project folder.
2. Run `npx serve` to start a local server.
3. Open the displayed URL in your browser.

## Technologies Used
- HTML5
- JavaScript (Phaser 3)
